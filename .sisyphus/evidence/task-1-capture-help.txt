usage: eufy_mqtt_client.py [-h] [--email EMAIL] [--password PASSWORD]
                           [--device DEVICE]
                           [--command {status,start,pause,dock,locate,quiet,standard,turbo,max}]
                           [--listen LISTEN] [--capture-dir CAPTURE_DIR]
                           [--duration DURATION]

Eufy X10 Pro Omni MQTT client

options:
  -h, --help            show this help message and exit
  --email EMAIL         Eufy account email (or EUFY_EMAIL env)
  --password PASSWORD   Eufy account password (or EUFY_PASSWORD env)
  --device DEVICE       Device serial number
  --command {status,start,pause,dock,locate,quiet,standard,turbo,max}
                        Command to execute
  --listen LISTEN       Seconds to listen for status (0=forever)
  --capture-dir CAPTURE_DIR
                        Directory to save captured HTTP responses and MQTT
                        messages as JSON fixtures. Creates http/ and mqtt/
                        subdirectories. Run anonymize_fixtures.py on the
                        output before committing.
  --duration DURATION   Capture duration in seconds (overrides --listen in
                        capture mode)
